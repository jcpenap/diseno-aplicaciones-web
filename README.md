# APIRest
## Dise√±o de aplicaciones Web

1. Download the repository
```
git clone https://github.com/jestrade/diseno-aplicaciones-web.git
```
2. Go to directory diseno-aplicaciones-web
```
cd diseno-aplicaciones-web
```
3. Install the app
```
npm install
```
4. Create .env file

```
ENV=development | production
DEV_DATABASE_HOST=
DEV_DATABASE_NAME=
PROD_DATABASE_HOST=
PROD_DATABASE_NAME=
PROD_DATABASE_USER=
PROD_DATABASE_PASSWORD=
SERVER_LOGS_DIR=
SERVER_PORT=
SERVER_HOST=
SECRET_KEY=
TOKEN_KEY=
```

5. To run the app on production mode
```
npm start
```

6. To run the app on development mode
6.1 Install nodemon
```
npm i nodemon -g
```
6.2 Run
```
npm run dev
```
7  To run tests
```
npm run test
```
